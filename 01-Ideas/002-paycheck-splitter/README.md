# 2. Paycheck Splitter

> Auto-divides irregular income into bills, savings, and spending

| Field | Detail |
|-------|--------|
| **Category** | Personal Finance |
| **Target Audience** | Working Class |
| **Monetization** | Freemium |
| **Recession Resistance** | High |
| **Build Effort** | Low - 6-8 weeks MVP |

## Problem

Gig workers and hourly employees have irregular income making budgeting nearly impossible. Traditional budgeting assumes bi-weekly paychecks.

## Solution

On each deposit, auto-split funds by percentage or fixed amounts into virtual envelopes. Priority-based: bills first, then savings, then discretionary.

## Market Size

$4.5B personal finance app market

## Existing Competitors

Qapital, Albert, YNAB

## Competitive Moat

Purpose-built for irregular income — not adapted from salary-based tools
