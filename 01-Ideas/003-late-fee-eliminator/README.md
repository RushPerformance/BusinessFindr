# 3. Late Fee Eliminator

> Monitors all bills and auto-pays minimums before late fees hit

| Field | Detail |
|-------|--------|
| **Category** | Personal Finance |
| **Target Audience** | Working Class |
| **Monetization** | Subscription |
| **Recession Resistance** | High |
| **Build Effort** | Medium - 10-12 weeks MVP |

## Problem

Late fees cost Americans $12B/year. One missed payment can trigger cascading fees and credit score damage.

## Solution

Aggregate all bills via email scanning and bank connections, send escalating reminders (7 days, 3 days, 1 day), auto-pay minimum if user hasn't paid by deadline.

## Market Size

$12B in annual late fees (addressable market)

## Existing Competitors

Prism, Doxo

## Competitive Moat

Proactive intervention model — others just display bills, this prevents fees
