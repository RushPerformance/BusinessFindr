# 4. Side Hustle Tax Tracker

> Snap receipts and track mileage for gig worker tax deductions

| Field | Detail |
|-------|--------|
| **Category** | Personal Finance |
| **Target Audience** | Working Class |
| **Monetization** | Freemium |
| **Recession Resistance** | High |
| **Build Effort** | Low - 6-8 weeks MVP |

## Problem

57M Americans do gig work. Most overpay taxes by $2,000-5,000/year because they miss deductions. Quarterly estimated taxes are confusing.

## Solution

Auto-track mileage via GPS, snap receipts for instant categorization, calculate quarterly estimated taxes in real time, generate Schedule C at tax time.

## Market Size

$1.2B tax prep software market (gig segment growing 25%/yr)

## Existing Competitors

Stride, Hurdlr, QuickBooks Self-Employed

## Competitive Moat

Built exclusively for gig workers — not simplified versions of business software
